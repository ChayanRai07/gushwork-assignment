# Gushwork Frontend Assignment

Developer: Chayan Rai

## Overview
This project is a frontend implementation based on the provided Figma design. The goal was to accurately recreate the layout, functionality, and responsiveness.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features Implemented
- Pixel-accurate UI based on Figma design
- Sticky header functionality
- Image carousel with zoom feature
- Fully responsive design (including 360px mobile screens)
- Clean and well-structured code

## Project Structure
`index.html` – Main page layout  
`styles.css` – Styling and responsive design  
`script.js` – JavaScript functionality (carousel, interactions)

## How to Run
1. Clone the repository
2. Open `index.html` in any browser
